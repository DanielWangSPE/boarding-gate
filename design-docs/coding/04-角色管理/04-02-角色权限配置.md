# 04-02 角色权限配置

## 1. 功能说明

本模块负责为角色分配功能权限（菜单 + 按钮）和数据权限（数据范围），是 RBAC 模型中角色与权限关联的核心操作。

## 2. 功能清单

### 2.1 功能权限分配

- 为角色分配菜单权限（勾选菜单树）
- 为角色分配按钮/操作权限
- 权限变更后实时生效（用户下次请求时使用新权限集）
- 权限分配界面以树形结构展示，支持全选/半选

### 2.2 数据权限配置

- 为角色配置数据权限范围
- 预设权限范围：全部数据 / 本部门数据 / 本部门及下级数据 / 仅本人数据
- 支持自定义数据权限（指定可访问的部门列表）

## 3. 业务规则

| 编号  | 规则         | 说明                                           |
| ----- | ------------ | ---------------------------------------------- |
| BR-01 | 权限实时生效 | 角色权限变更后，关联用户的权限缓存立即清除     |
| BR-02 | 子应用隔离   | 子应用角色仅可分配该子应用下的菜单权限         |
| BR-03 | 超管豁免     | 超级管理员角色拥有全部权限，不通过菜单分配控制 |
| BR-04 | 半选状态     | 父菜单下仅部分子菜单被选中时，父菜单为半选状态 |

## 4. 接口要求

| 接口                    | 方法 | 说明                                       |
| ----------------------- | ---- | ------------------------------------------ |
| /roles/{id}/permissions | GET  | 获取角色的功能权限（已勾选的菜单 ID 列表） |
| /roles/{id}/permissions | PUT  | 分配角色功能权限（全量替换）               |
| /roles/{id}/data-scope  | GET  | 获取角色数据权限配置                       |
| /roles/{id}/data-scope  | PUT  | 配置角色数据权限                           |

## 5. 页面清单

| 页面               | 说明                                        |
| ------------------ | ------------------------------------------- |
| 角色权限配置页     | 树形菜单勾选界面，支持全选/半选/展开/折叠   |
| 角色数据权限配置页 | 选择数据权限范围 + 自定义部门选择（穿梭框） |
