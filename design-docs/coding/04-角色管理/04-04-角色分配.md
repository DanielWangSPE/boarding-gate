# 04-04 角色分配

## 1. 功能说明

本模块负责从角色维度管理角色与用户的关联关系，包括查看角色下的用户、批量添加/移除用户、按组织批量分配等。

> 注：用户维度的角色分配由用户管理模块（02-03 用户关联关系）承担，本模块为角色维度的操作。

## 2. 功能清单

### 2.1 角色用户管理

- 角色维度：查看角色下的所有用户，批量添加/移除用户
- 支持按组织批量分配角色（如为某部门全员分配指定角色）

### 2.2 角色有效期（扩展）

- 支持角色有效期（临时角色，到期自动回收）

> 角色有效期功能一期预留数据模型，不实现定时回收逻辑。

## 3. 业务规则

| 编号  | 规则         | 说明                                         |
| ----- | ------------ | -------------------------------------------- |
| BR-01 | 停用角色限制 | 不可向已停用的角色添加用户                   |
| BR-02 | 内置角色限制 | 超级管理员角色仅允许超级管理员操作其用户关联 |
| BR-03 | 权限刷新     | 添加/移除用户后，相关用户的权限缓存立即清除  |

## 4. 接口要求

| 接口                      | 方法   | 说明                         |
| ------------------------- | ------ | ---------------------------- |
| /roles/{id}/users         | GET    | 查询角色下的用户列表（分页） |
| /roles/{id}/users         | POST   | 批量向角色添加用户           |
| /roles/{id}/users         | DELETE | 批量从角色移除用户           |
| /roles/{id}/users/by-dept | POST   | 按部门批量分配角色           |

## 5. 页面清单

| 页面           | 说明                                       |
| -------------- | ------------------------------------------ |
| 角色用户列表页 | 查看角色关联用户，支持添加/移除/按部门分配 |
