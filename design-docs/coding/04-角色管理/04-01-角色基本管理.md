# 04-01 角色基本管理

## 1. 功能说明

本模块负责角色信息的定义和维护，包括角色的增删改查、状态管理和内置角色保护。

## 2. 功能清单

### 2.1 角色信息

- 角色名称
- 角色标识（唯一编码，用于程序层面权限判断）
- 角色分类（系统角色 / 业务角色 / 子应用角色）
- 角色所属应用（全局 / 指定子应用）
- 角色描述
- 角色状态（正常 / 停用）
- 排序号
- 数据权限范围（全部 / 自定义 / 本部门 / 本部门及下级 / 仅本人）

### 2.2 角色操作

- 角色的增删改查
- 角色停用后，已关联该角色的用户对应权限立即失效
- 删除角色前需确认无用户关联
- 内置系统角色（如超级管理员）不可删除和修改标识

## 3. 业务规则

| 编号  | 规则         | 说明                                       |
| ----- | ------------ | ------------------------------------------ |
| BR-01 | 角色标识唯一 | 角色标识全局唯一，创建后不可修改           |
| BR-02 | 内置角色保护 | 超级管理员等内置角色不可删除、不可修改标识 |
| BR-03 | 删除前置检查 | 角色被用户引用时不可直接删除               |
| BR-04 | 停用即失效   | 角色停用后，关联用户的权限缓存立即清除     |
| BR-05 | 角色名称唯一 | 同一所属应用下角色名称不可重复             |

## 4. 接口要求

| 接口               | 方法   | 说明             |
| ------------------ | ------ | ---------------- |
| /roles             | GET    | 分页查询角色列表 |
| /roles             | POST   | 创建角色         |
| /roles/{id}        | GET    | 获取角色详情     |
| /roles/{id}        | PUT    | 更新角色         |
| /roles/{id}        | DELETE | 删除角色         |
| /roles/{id}/status | PUT    | 启用/停用角色    |

## 5. 页面清单

| 页面            | 说明                                   |
| --------------- | -------------------------------------- |
| 角色列表页      | 角色列表，支持按分类/状态/所属应用筛选 |
| 角色编辑页/弹窗 | 新增/编辑角色基本信息                  |
