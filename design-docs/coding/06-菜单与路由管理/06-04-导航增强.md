# 06-04 导航增强

## 1. 功能说明

本模块提供门户平台的导航增强能力，包括应用导航栏（门户/子应用切换）、多标签页、面包屑导航和全局菜单搜索，提升用户在多应用场景下的使用体验。子应用内部导航结构由子应用自行管理，本模块仅负责门户层的导航展示与跨应用联动。

## 2. 功能清单

### 2.1 应用导航栏

- 顶部或侧边提供应用切换入口
- 展示当前所在应用（门户 / 子应用名称）
- 点击可快速切换到其他子应用或返回门户首页
- 导航栏中的子应用列表与用户权限一致（同 `/apps/user` 接口结果）

### 2.2 多标签页（Tab 导航）

- 支持多标签页模式，用户可在多个页面间切换
- 标签页支持关闭当前、关闭其他、关闭所有、关闭左侧/右侧
- 标签页支持固定（固定的标签不可关闭）
- 跨子应用的标签页统一管理
- 切换子应用时保留其他子应用的标签页状态
- 刷新页面后恢复之前打开的标签页

### 2.3 面包屑导航

- 根据菜单层级自动生成面包屑
- 子应用页面的面包屑至少包含应用名前缀，子应用内部层级由子应用自行补充
- 面包屑支持点击跳转

### 2.4 菜单搜索

- 支持全局菜单搜索（快捷键触发，如 Ctrl + K）
- 搜索范围包含门户菜单和用户有权访问的子应用入口
- 按拼音或关键字匹配菜单名称
- 搜索结果点击可直接跳转到门户页面或子应用入口

## 3. 业务规则

| 编号  | 规则         | 说明                                                 |
| ----- | ------------ | ---------------------------------------------------- |
| BR-01 | 标签页持久化 | 刷新页面后恢复之前打开的标签页（sessionStorage）     |
| BR-02 | 跨应用标签   | 不同子应用的标签页统一管理，标签上显示应用标识       |
| BR-03 | 固定标签     | 固定的标签不可关闭，位于标签栏最左侧                 |
| BR-04 | 搜索范围     | 菜单搜索仅搜索门户菜单和用户有权访问的子应用入口     |
| BR-05 | 面包屑层级   | 子应用面包屑至少包含应用名前缀，内部层级由子应用补充 |

## 4. 页面清单

| 页面/组件      | 说明                   |
| -------------- | ---------------------- |
| 应用导航栏组件 | 顶部应用切换导航       |
| 标签页栏组件   | 页面标签页管理         |
| 面包屑组件     | 层级导航               |
| 菜单搜索弹窗   | 全局搜索菜单并快速跳转 |
