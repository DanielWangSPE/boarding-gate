# 03-02 部门成员管理

## 1. 功能说明

本模块负责管理部门与用户之间的归属关系，包括查看部门成员、设置用户所属部门、设置部门负责人等操作。

## 2. 功能清单

### 2.1 成员查看

- 查看部门下的直属成员
- 查看部门下的所有成员（含下级部门）
- 成员列表支持搜索和分页

### 2.2 成员管理

- 将用户设置到指定部门
- 调整用户所属部门时，原部门归属自动替换
- 支持批量设置用户所属部门

### 2.3 部门负责人

- 设置/变更部门负责人（从部门成员中选择）
- 部门负责人可用于审批流程和数据权限判断

## 3. 业务规则

| 编号  | 规则         | 说明                                     |
| ----- | ------------ | ---------------------------------------- |
| BR-01 | 单一归属     | 每个用户仅归属一个组织节点               |
| BR-02 | 负责人范围   | 部门负责人必须是该部门的成员             |
| BR-03 | 移除负责人   | 调整负责人所属部门前需先取消其负责人身份 |
| BR-04 | 停用部门限制 | 不可将用户归属到已停用的部门             |

## 4. 接口要求

| 接口                      | 方法 | 说明                 |
| ------------------------- | ---- | -------------------- |
| /departments/{id}/members | GET  | 获取部门成员列表     |
| /departments/{id}/members | POST | 将用户设置到当前部门 |
| /departments/{id}/leader  | PUT  | 设置部门负责人       |

## 5. 页面清单

| 页面                | 说明                                 |
| ------------------- | ------------------------------------ |
| 组织架构页-成员列表 | 右侧面板展示选中部门的成员列表       |
| 设置成员归属弹窗    | 从用户列表中选择用户并设置到当前部门 |
