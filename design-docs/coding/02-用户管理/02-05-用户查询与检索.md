# 02-05 用户查询与检索

## 1. 功能说明

本模块提供用户数据的多条件查询、筛选和分页能力，支撑用户列表页面和其他模块的用户选择场景。

## 2. 功能清单

### 2.1 查询能力

- 支持按用户名、姓名、手机号、邮箱模糊搜索
- 支持按组织部门筛选（含下级部门）
- 支持按角色筛选
- 支持按用户状态筛选
- 支持按用户类型筛选
- 支持按创建时间范围筛选
- 用户列表分页展示，支持自定义每页条数

### 2.2 数据权限过滤

- 根据当前用户的数据权限范围自动过滤结果
- 全部数据权限：可查看所有用户
- 本部门权限：仅可查看本部门用户
- 本部门及下级权限：可查看本部门及下级部门用户

## 3. 业务规则

| 编号  | 规则           | 说明                                     |
| ----- | -------------- | ---------------------------------------- |
| BR-01 | 默认排序       | 默认按创建时间倒序                       |
| BR-02 | 排除注销       | 查询默认排除已注销用户，管理员可选择包含 |
| BR-03 | 数据权限       | 查询结果自动受数据权限范围约束           |
| BR-04 | 部门筛选含下级 | 按部门筛选时默认包含该部门的所有下级部门 |

## 4. 接口要求

| 接口 | 方法 | 说明 |
| --- | --- | --- |
| /users | GET | 分页查询用户列表 |
| /users/select | GET | 用户选择器（轻量接口，仅返回 id + 名称，用于下拉选择） |

## 5. 页面清单

| 页面       | 说明                                             |
| ---------- | ------------------------------------------------ |
| 用户列表页 | 展示用户列表，支持搜索、筛选、排序、批量操作入口 |
