# 02-04 批量管理

## 1. 功能说明

本模块提供用户状态、角色、部门和密码相关的批量管理能力，提升管理员的操作效率。

> 当前阶段不纳入用户批量导入、批量导出能力。

## 2. 功能清单

### 2.1 批量管理

- 批量启用/停用用户
- 批量分配角色
- 批量调整部门
- 批量重置密码

## 3. 业务规则

| 编号  | 规则       | 说明                                         |
| ----- | ---------- | -------------------------------------------- |
| BR-01 | 批量保护   | 批量操作中自动跳过超级管理员和当前操作者自身 |
| BR-02 | 结果独立   | 批量操作中单个用户失败不影响其他用户处理     |
| BR-03 | 部门有效性 | 批量调整部门时不可设置到已停用部门           |
| BR-04 | 角色有效性 | 批量分配角色时不可分配已停用角色             |

## 4. 接口要求

| 接口                        | 方法 | 说明          |
| --------------------------- | ---- | ------------- |
| /users/batch/status         | PUT  | 批量启用/停用 |
| /users/batch/roles          | PUT  | 批量分配角色  |
| /users/batch/departments    | PUT  | 批量调整部门  |
| /users/batch/reset-password | POST | 批量重置密码  |

## 5. 页面清单

本模块无独立页面，通过用户列表页的批量操作入口触发。
