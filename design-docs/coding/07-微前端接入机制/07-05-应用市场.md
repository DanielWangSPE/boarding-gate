# 07-05 应用市场（扩展）

## 1. 功能说明

本模块为扩展功能，提供子应用的展示、浏览和订阅能力。用户可在应用市场中查看所有可用应用并自主管理个人应用列表。管理员可查看应用使用统计数据。

## 2. 功能清单

### 2.1 应用展示

- 展示所有已注册且状态正常的子应用列表
- 子应用的简介、截图、使用说明
- 按分类筛选和搜索

### 2.2 应用订阅

- 用户可自主订阅/取消订阅子应用
- 订阅的应用出现在应用导航栏和相关应用入口中
- 取消订阅不影响用户的角色权限，仅影响个人菜单展示
- 管理员强制分配的应用不可取消订阅

### 2.3 使用统计

- 子应用的活跃用户数
- 子应用的日/周/月访问量
- 子应用的使用时长统计

## 3. 业务规则

| 编号  | 规则      | 说明                               |
| ----- | --------- | ---------------------------------- |
| BR-01 | 权限前提  | 用户只能订阅有权限访问的子应用     |
| BR-02 | 强制应用  | 管理员指定的必装应用不可取消订阅   |
| BR-03 | 订阅≠权限 | 订阅仅控制个人菜单展示，不影响权限 |

## 4. 接口要求

| 接口                      | 方法 | 说明               |
| ------------------------- | ---- | ------------------ |
| /app-market               | GET  | 获取应用市场列表   |
| /app-market/subscribe     | POST | 订阅应用           |
| /app-market/unsubscribe   | POST | 取消订阅           |
| /app-market/my            | GET  | 获取我的已订阅应用 |
| /app-market/stats/{appId} | GET  | 获取应用使用统计   |

## 5. 页面清单

| 页面       | 说明                   |
| ---------- | ---------------------- |
| 应用市场页 | 浏览、搜索、订阅子应用 |
| 我的应用页 | 管理个人已订阅的应用   |
