# 08-04 文件服务

## 1. 功能说明

本模块为其他业务模块提供基础的文件上传/下载能力，支撑用户头像上传、子应用图标等场景。一期仅提供 API 服务，不做独立的文件管理页面。

## 2. 功能清单

### 2.1 一期功能

- 文件上传（单文件，限制大小和类型）
- 文件下载
- 文件删除
- 存储后端：本地存储或 MinIO（按部署环境选择）

### 2.2 二期扩展

- 大文件分片上传
- 文件预览（图片、PDF、Office 文档）
- 多存储后端切换（OSS、S3）
- 文件管理页面（列表、统计）

## 3. 业务规则

| 编号  | 规则     | 说明                                                      |
| ----- | -------- | --------------------------------------------------------- |
| BR-01 | 类型校验 | 文件上传需校验文件类型（白名单），防止恶意文件            |
| BR-02 | 大小限制 | 文件大小不超过系统参数 `file.upload.maxSize`（默认 10MB） |
| BR-03 | 存储路径 | 文件按 `年/月/日` 目录结构存储，文件名使用 UUID 重命名    |
| BR-04 | 访问控制 | 文件下载需要登录态，通过文件 ID 获取                      |

## 4. 接口要求

| 接口          | 方法   | 说明                            |
| ------------- | ------ | ------------------------------- |
| /files/upload | POST   | 上传文件（multipart/form-data） |
| /files/{id}   | GET    | 下载文件                        |
| /files/{id}   | DELETE | 删除文件                        |
