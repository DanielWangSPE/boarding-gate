# 08-01 数据字典 — 编码规范（Spec）

## 1. 数据模型

### 1.1 字典类型表（sys_dict_type）

```sql
CREATE TABLE sys_dict_type (
  id          BIGINT       NOT NULL AUTO_INCREMENT COMMENT '主键',
  dict_name   VARCHAR(128) NOT NULL                 COMMENT '字典名称',
  dict_code   VARCHAR(128) NOT NULL                 COMMENT '字典编码',
  status      TINYINT(1)   NOT NULL DEFAULT 1       COMMENT '状态：1=正常, 0=停用',
  is_builtin  TINYINT(1)   NOT NULL DEFAULT 0       COMMENT '是否内置：1=是, 0=否',
  remark      VARCHAR(256) DEFAULT NULL             COMMENT '备注',
  created_by  BIGINT       DEFAULT NULL,
  created_at  DATETIME     NOT NULL DEFAULT CURRENT_TIMESTAMP,
  updated_by  BIGINT       DEFAULT NULL,
  updated_at  DATETIME     NOT NULL DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
  PRIMARY KEY (id),
  UNIQUE INDEX uk_dict_code (dict_code)
) COMMENT '字典类型表';
```

### 1.2 字典数据表（sys_dict_data）

```sql
CREATE TABLE sys_dict_data (
  id          BIGINT       NOT NULL AUTO_INCREMENT COMMENT '主键',
  dict_code   VARCHAR(128) NOT NULL                 COMMENT '所属字典编码',
  dict_label  VARCHAR(128) NOT NULL                 COMMENT '字典标签（展示文本）',
  dict_value  VARCHAR(128) NOT NULL                 COMMENT '字典值（存储值）',
  sort_order  INT          NOT NULL DEFAULT 0       COMMENT '排序号',
  css_class   VARCHAR(64)  DEFAULT NULL             COMMENT '样式属性（primary/success/warning/danger）',
  is_default  TINYINT(1)   NOT NULL DEFAULT 0       COMMENT '是否默认：1=是, 0=否',
  status      TINYINT(1)   NOT NULL DEFAULT 1       COMMENT '状态：1=正常, 0=停用',
  remark      VARCHAR(256) DEFAULT NULL             COMMENT '备注',
  created_by  BIGINT       DEFAULT NULL,
  created_at  DATETIME     NOT NULL DEFAULT CURRENT_TIMESTAMP,
  updated_by  BIGINT       DEFAULT NULL,
  updated_at  DATETIME     NOT NULL DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
  PRIMARY KEY (id),
  UNIQUE INDEX uk_dict_value (dict_code, dict_value),
  INDEX idx_dict_code (dict_code)
) COMMENT '字典数据表';
```

## 2. 接口规范

### 2.1 按字典编码获取字典项（前端使用）

```
GET /dict/data/{code}
Authorization: Bearer {accessToken}
```

**Response 200**

```json
{
  "code": 200,
  "data": [
    { "label": "男", "value": "0", "cssClass": null, "isDefault": false },
    { "label": "女", "value": "1", "cssClass": null, "isDefault": false },
    { "label": "未知", "value": "2", "cssClass": null, "isDefault": false }
  ]
}
```

**处理流程**

```
查询字典项
  │
  ├─ 1. 查询 Redis 缓存 dict:{code}
  │      └─ 命中 → 直接返回
  ├─ 2. 从 DB 查询
  │      ├─ 校验字典类型存在且 status=1
  │      └─ SELECT * FROM sys_dict_data WHERE dict_code = ? AND status = 1 ORDER BY sort_order
  ├─ 3. 写入 Redis 缓存
  └─ 4. 返回
```

### 2.2 创建字典类型

```
POST /dict/types
```

**处理流程**

```
创建字典类型
  │
  ├─ 1. dict_code 唯一校验
  ├─ 2. INSERT sys_dict_type
  ├─ 3. 记录操作留痕
  └─ 4. 返回
```

### 2.3 更新/删除字典数据

```
PUT /dict/data/{id}
DELETE /dict/data/{id}
```

**处理流程**（更新/删除/新增字典数据时均执行）

```
字典数据变更
  │
  ├─ 1. 校验字典类型存在
  ├─ 2. 内置字典类型检查（删除类型时拒绝）
  ├─ 3. 同类型下 dict_value 唯一校验（新增/更新时）
  ├─ 4. 执行数据库操作
  ├─ 5. 清除该字典的 Redis 缓存 → DEL dict:{code}
  ├─ 6. 记录操作留痕
  └─ 7. 返回
```

## 3. 缓存设计

| Key              | 内容                          | TTL                      |
| ---------------- | ----------------------------- | ------------------------ |
| `dict:{code}`    | 指定字典编码的字典项列表 JSON | 不过期（变更时主动清除） |
| `dict:all-codes` | 所有字典编码列表              | 60 分钟                  |

**刷新时机**：字典数据增删改时清除对应 `dict:{code}`。

## 4. 前端字典工具

### 4.1 字典 Store

```typescript
const useDictStore = defineStore('dict', {
  state: () => ({
    cache: new Map<string, DictItem[]>(),
  }),

  actions: {
    async getDict(code: string): Promise<DictItem[]> {
      if (this.cache.has(code)) {
        return this.cache.get(code)!;
      }
      const res = await api.get(`/dict/data/${code}`);
      this.cache.set(code, res.data);
      return res.data;
    },

    clearCache(code?: string) {
      if (code) {
        this.cache.delete(code);
      } else {
        this.cache.clear();
      }
    },
  },
});
```

### 4.3 字典选择组件

```vue
<!-- DictSelect.vue -->
<template>
  <el-select v-model="modelValue" v-bind="$attrs">
    <el-option
      v-for="item in options"

      :label="item.label"
      :value="item.value"
    />
  </el-select>
```

### 4.4 字典标签组件

```vue
<!-- DictTag.vue：显示带颜色的字典标签 -->
<template>
  <el-tag v-if="item" :type="item.cssClass || ''">{{ item.label }}</el-tag>
  <span v-else>{{ value }}</span>
</template>

<script setup>
const props = defineProps<{ dictCode: string; value: string }>()
const dictStore = useDictStore()
const item = ref<DictItem | null>(null)

  const items = await dictStore.getDict(props.dictCode)
  item.value = items.find(i => i.value === props.value) || null
})
</script>
```

## 5. 错误码

| 错误码 | 说明               |
| ------ | ------------------ |
| A0800  | 字典类型不存在     |
| A0801  | 字典编码已存在     |
| A0802  | 内置字典不可删除   |
| A0803  | 同类型下字典值重复 |
| A0804  | 字典数据不存在     |

## 6. 安全要点

| 项目       | 要求                                                        |
| ---------- | ----------------------------------------------------------- |
| 权限控制   | 字典管理需 `admin:dict:manage` 权限，查询字典项无需特殊权限 |
| 内置保护   | `is_builtin=1` 的字典类型不可删除                           |
| 编码不可变 | `dict_code` 创建后不可修改                                  |
| 缓存一致性 | 字典变更后必须清除缓存                                      |
| 审计追踪   | 记录字典的创建、修改、删除操作                              |
