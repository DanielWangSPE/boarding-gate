# 08-02 系统参数配置

## 1. 功能说明

本模块负责管理门户平台的运行时配置参数，如验证码开关、密码策略、Token 有效期等。通过集中化的参数管理，避免将配置散落在代码或配置文件中，支持运行时动态调整。

## 2. 功能清单

### 2.1 参数管理

- 参数名称
- 参数键名（唯一标识，如 `sys.account.captchaEnabled`）
- 参数键值
- 参数类型（系统内置 / 用户自定义）
- 参数描述
- 系统内置参数不可删除，仅允许修改值

### 2.2 常见系统参数

| 参数键名                     | 默认值 | 说明                        |
| ---------------------------- | ------ | --------------------------- |
| `auth.captcha.enabled`       | true   | 是否开启验证码              |
| `auth.login.maxRetryCount`   | 5      | 密码错误锁定次数            |
| `auth.login.lockDuration`    | 30     | 账号锁定时长（分钟）        |
| `auth.token.accessExpire`    | 30     | Access Token 有效期（分钟） |
| `auth.token.refreshExpire`   | 7      | Refresh Token 有效期（天）  |
| `auth.session.idleTimeout`   | 60     | 会话空闲超时（分钟）        |
| `auth.password.minLength`    | 8      | 密码最小长度                |
| `auth.password.historyCount` | 1      | 密码历史记录数              |
| `auth.password.maxAgeDays`   | 90     | 密码过期天数                |
| `file.upload.maxSize`        | 10     | 文件上传最大大小（MB）      |

## 3. 业务规则

| 编号  | 规则       | 说明                                     |
| ----- | ---------- | ---------------------------------------- |
| BR-01 | 键名唯一   | 参数键名全局唯一                         |
| BR-02 | 内置保护   | 系统内置参数不可删除，仅允许修改值       |
| BR-03 | 即时生效   | 参数修改后需刷新缓存，下次读取时即时生效 |
| BR-04 | 值类型校验 | 数字类型参数需校验值为合法数字           |

## 4. 接口要求

| 接口              | 方法   | 说明                               |
| ----------------- | ------ | ---------------------------------- |
| /config           | GET    | 查询系统参数列表（分页，支持搜索） |
| /config           | POST   | 创建系统参数                       |
| /config/{id}      | PUT    | 更新系统参数                       |
| /config/{id}      | DELETE | 删除系统参数                       |
| /config/key/{key} | GET    | 按键名获取参数值（内部调用）       |

## 5. 页面清单

| 页面       | 说明                             |
| ---------- | -------------------------------- |
| 参数设置页 | 系统参数列表，支持搜索和在线编辑 |
