# 08-02 系统参数配置 — 编码规范（Spec）

## 1. 数据模型

### 1.1 系统参数表（sys_config）

```sql
CREATE TABLE sys_config (
  id            BIGINT       NOT NULL AUTO_INCREMENT COMMENT '主键',
  config_name   VARCHAR(128) NOT NULL                 COMMENT '参数名称',
  config_key    VARCHAR(256) NOT NULL                 COMMENT '参数键名',
  config_value  VARCHAR(512) NOT NULL                 COMMENT '参数键值',
  config_type   TINYINT(1)   NOT NULL DEFAULT 0       COMMENT '类型：1=内置, 0=自定义',
  remark        VARCHAR(256) DEFAULT NULL             COMMENT '备注',
  created_by    BIGINT       DEFAULT NULL,
  created_at    DATETIME     NOT NULL DEFAULT CURRENT_TIMESTAMP,
  updated_by    BIGINT       DEFAULT NULL,
  updated_at    DATETIME     NOT NULL DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
  PRIMARY KEY (id),
  UNIQUE INDEX uk_config_key (config_key)
) COMMENT '系统参数表';
```

## 2. 接口规范

### 2.1 查询参数列表

```
GET /config?keyword={keyword}&configType={configType}&page={page}&pageSize={pageSize}
Authorization: Bearer {accessToken}
```

### 2.2 按键名获取参数值

```
GET /config/key/{key}
Authorization: Bearer {accessToken}
```

**Response 200**

```json
{
  "code": 200,
  "data": {
    "configKey": "auth.login.maxRetryCount",
    "configValue": "5"
  }
}
```

### 2.3 创建参数

```
POST /config
```

**处理流程**

```
创建参数
  │
  ├─ 1. config_key 唯一校验
  ├─ 2. INSERT sys_config（config_type = 0，用户自定义）
  ├─ 3. 清除参数缓存
  ├─ 4. 记录操作留痕
  └─ 5. 返回
```

### 2.4 更新参数

```
PUT /config/{id}
```

**处理流程**

```
更新参数
  │
  ├─ 1. 校验参数存在
  ├─ 2. 内置参数仅允许修改 config_value 和 remark
  ├─ 3. config_key 不可修改
  ├─ 4. UPDATE sys_config
  ├─ 5. 清除参数缓存 → DEL config:{key}
  ├─ 6. 记录操作留痕（含变更前后值）
  └─ 7. 返回
```

### 2.5 删除参数

```
DELETE /config/{id}
```

**处理流程**

```
删除参数
  │
  ├─ 1. 校验参数存在
  ├─ 2. 内置参数（config_type=1）拒绝删除
  ├─ 3. DELETE sys_config
  ├─ 4. 清除参数缓存
  ├─ 5. 记录操作留痕
  └─ 6. 返回
```

## 3. 缓存设计

| Key            | 内容         | TTL                      |
| -------------- | ------------ | ------------------------ |
| `config:{key}` | 参数值字符串 | 不过期（变更时主动清除） |

**刷新时机**：参数增删改时清除 `config:{key}`。

## 4. 后端参数读取工具

### 4.1 ConfigService

```java
@Service
public class ConfigService {

    @Autowired
    private RedisTemplate<String, String> redisTemplate;
    @Autowired
    private SysConfigMapper configMapper;

    public String getValue(String key) {
        String cacheKey = "config:" + key;
        String value = redisTemplate.opsForValue().get(cacheKey);
        if (value != null) return value;

        SysConfig config = configMapper.selectByKey(key);
        if (config == null) return null;

        redisTemplate.opsForValue().set(cacheKey, config.getConfigValue());
        return config.getConfigValue();
    }

    public int getIntValue(String key, int defaultValue) {
        String value = getValue(key);
        if (value == null) return defaultValue;
        try {
            return Integer.parseInt(value);
        } catch (NumberFormatException e) {
            return defaultValue;
        }
    }

    public boolean getBoolValue(String key, boolean defaultValue) {
        String value = getValue(key);
        if (value == null) return defaultValue;
        return Boolean.parseBoolean(value);
    }
}
```

### 4.2 业务模块使用示例

```java
// 登录模块读取锁定阈值
int maxRetry = configService.getIntValue("auth.login.maxRetryCount", 5);
int lockMinutes = configService.getIntValue("auth.login.lockDuration", 30);

// 验证码模块读取开关
boolean captchaEnabled = configService.getBoolValue("auth.captcha.enabled", true);
```

## 5. 前端参数页面

### 5.1 参数列表页

```
参数列表
  ├─ 搜索栏：关键词搜索（名称/键名）、类型筛选（内置/自定义）
  ├─ 表格列：参数名称、键名、键值、类型（标签）、备注、操作
  ├─ 操作列：
  │      ├─ 编辑（弹窗内编辑值和备注）
  │      └─ 删除（仅自定义参数显示）
  └─ 内置参数行以特殊标记区分（如标签或背景色）
```

### 5.2 参数编辑

```
编辑弹窗
  ├─ 参数名称（内置参数只读）
  ├─ 参数键名（只读）
  ├─ 参数键值（可编辑，根据参数类型提供不同输入方式）
  │      ├─ 布尔值 → Switch 开关
  │      ├─ 数字 → InputNumber
  │      └─ 字符串 → Input
  └─ 备注
```

## 6. 错误码

| 错误码 | 说明             |
| ------ | ---------------- |
| A0810  | 参数不存在       |
| A0811  | 参数键名已存在   |
| A0812  | 内置参数不可删除 |
| A0813  | 参数键名不可修改 |
| A0814  | 参数值格式无效   |

## 7. 安全要点

| 项目 | 要求 |
| --- | --- |
| 权限控制 | 参数管理需 `admin:config:manage` 权限；按键名查询仅限内部服务调用 |
| 内置保护 | 内置参数不可删除、不可改键名 |
| 值校验 | 数字类型参数校验值合法性和范围 |
| 审计追踪 | 记录参数变更的前后值，便于追溯 |
| 缓存一致性 | 参数变更后清除缓存 |
