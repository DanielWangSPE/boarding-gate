# 02 - 用户管理

## 1. 模块概述

用户管理模块负责系统用户的全生命周期管理，从创建、启用、变更到停用/注销，并提供用户信息维护、查询检索、批量管理以及用户与组织/角色关联能力。作为统一门户的核心模块，本模块管理的用户数据将为所有子应用提供统一的用户身份基础。

## 3. 业务规则汇总

| 编号 | 规则 | 说明 | 所属模块 |
| --- | --- | --- | --- |
| BR-01 | 用户名唯一 | 用户名全局唯一，不可重复（注销释放后可复用） | 用户信息管理 |
| BR-02 | 手机号唯一 | 手机号非必填，但填写时同一用户类型下不可重复 | 用户信息管理 |
| BR-03 | 邮箱唯一 | 邮箱非必填，但填写时同一用户类型下不可重复 | 用户信息管理 |
| BR-04 | 停用即踢 | 用户被停用后，所有活跃会话立即失效 | 用户生命周期 |
| BR-05 | 超级管理员保护 | 超级管理员账号不可被停用或删除 | 用户生命周期 |
| BR-06 | 自我保护 | 用户不可停用或删除自己的账号 | 用户生命周期 |
| BR-07 | 初始密码 | 系统生成的初始密码须符合密码复杂度策略 | 用户生命周期 |

## 4. 与其他模块的关系

| 关联模块 | 关系说明 |
| --- | --- |
| 认证与会话管理 | 提供用户账号信息用于登录验证，并在停用/注销、管理员重置密码等场景下执行会话失效 |
| 组织架构管理 | 用户归属于组织部门 |
| 角色管理 | 用户被分配角色 |
| 权限管理 | 通过角色间接获取权限 |
