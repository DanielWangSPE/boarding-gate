# 09 - 门户首页与个人中心

## 1. 模块概述

门户首页是用户登录后的统一首页入口，当前阶段仅保留欢迎页能力，不承载待办、快捷入口、公告聚合或数据看板等内容。个人中心为用户提供个人信息维护和基础安全设置能力，是该模块当前阶段的核心建设内容。

## 2. 功能清单

### 2.1 门户首页

#### 2.1.1 欢迎页

- 门户首页保留为用户登录后的统一首页入口
- 页面仅展示欢迎文案和基础品牌信息
- 不展示待办事项、快捷入口、公告栏、应用卡片、数据看板等业务内容
- 后续如需扩展首页能力，再单独补充需求设计

### 2.2 个人中心

#### 2.2.1 个人信息

- 查看和编辑个人基本信息（姓名、头像、邮箱、手机号等）
- 修改手机号（需验证码校验）
- 修改邮箱（需验证码校验）
- 上传/更换头像

#### 2.2.2 安全设置

- 修改密码

### 2.3 全局搜索

- 支持全局搜索功能（快捷键 Ctrl+K 或顶部搜索框）
- 搜索范围：菜单名称、用户名称、快捷指令
- 搜索结果分类展示
- 搜索历史记录

## 3. 业务规则

| 编号  | 规则     | 说明                                           |
| ----- | -------- | ---------------------------------------------- |
| BR-01 | 绑定保护 | 用户不可解绑唯一的登录凭证                     |
| BR-02 | 首页范围 | 当前阶段门户首页仅作为统一首页入口和欢迎页使用 |

## 4. 接口要求

| 接口                  | 方法 | 说明                                       |
| --------------------- | ---- | ------------------------------------------ |
| /profile              | GET  | 获取个人信息                               |
| /profile              | PUT  | 更新个人信息                               |
| /profile/avatar       | POST | 上传头像                                   |
| /auth/password/change | POST | 修改密码（由个人中心页面调用认证模块接口） |
| /profile/phone        | PUT  | 修改手机号                                 |
| /profile/email        | PUT  | 修改邮箱                                   |
| /search               | GET  | 全局搜索                                   |

## 5. 页面清单

| 页面       | 说明                                 |
| ---------- | ------------------------------------ |
| 门户首页   | 登录后的统一首页入口，仅展示欢迎文案 |
| 个人信息页 | 查看和编辑个人基本信息               |
| 安全设置页 | 密码修改                             |

## 6. 与其他模块的关系

| 关联模块 | 关系说明 |
| --- | --- |
| 认证与会话管理 | 修改密码 |
| 用户管理 | 个人信息来源于用户管理数据 |
| 菜单与导航管理 | 全局搜索依赖门户菜单与应用入口数据，门户首页路由作为统一首页入口 |
