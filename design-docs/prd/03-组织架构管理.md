# 03 - 组织架构管理

## 1. 模块概述

组织架构管理模块负责维护公司的组织树结构和用户归属关系，提供部门树管理、部门成员归属调整和部门负责人设置能力。当前范围仅关注组织节点之间的父子关系，以及每个用户所属的组织节点。

## 3. 业务规则汇总

| 编号  | 规则         | 说明                                 | 所属模块 |
| ----- | ------------ | ------------------------------------ | -------- |
| BR-01 | 根部门保护   | 根部门（公司）不可删除和停用         | 部门管理 |
| BR-02 | 部门编码唯一 | 部门编码全局唯一，创建后不可修改     | 部门管理 |
| BR-03 | 删除限制     | 部门下存在用户或子部门时不可直接删除 | 部门管理 |
| BR-04 | 停用级联     | 停用部门时，其下所有子部门同步停用   | 部门管理 |
